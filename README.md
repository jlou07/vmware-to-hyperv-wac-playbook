# VMware to Hyper-V migration using Windows Admin Center

This repository provides a practical playbook to migrate virtual machines
from VMware to Hyper-V using the VM Conversion extension in Windows Admin Center.

## Context

Following recent VMware licensing changes, many partners and customers
are looking for concrete exit scenarios.

This approach allows:
- A fast and controlled exit from VMware
- Minimal downtime
- An on-premises first step toward a hybrid or cloud-ready strategy

## Scope

This documentation focuses on:
- When this approach makes sense
- Required prerequisites
- Migration workflow using Windows Admin Center
- Limitations and known constraints
- Positioning as part of a hybrid or Azure journey
