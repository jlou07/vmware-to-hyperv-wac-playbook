# Prerequisites

This section translates technical requirements into partner pre-checks for presales qualification and delivery readiness.

## Windows Admin Center

- Windows Admin Center (latest available version)
- VM Conversion extension enabled
- Stakeholder awareness that VM Conversion is Preview

## VMware Source Environment

- vCenter 6.x or 7.x
- No active snapshots on VMs targeted for migration
- Guest OS is supported (Windows or Linux)
- Confirm source VM inventory and migration wave grouping

## Hyper-V Target Environment

- Hyper-V role installed and validated
- Sufficient CPU, RAM, and storage for landing zone capacity
- Baseline host health checks completed before migration windows

## Delivery Gating Checklist

- Source and target ownership confirmed
- Change window and rollback criteria approved
- Pilot VM list approved by customer
- Business-critical workloads prioritized by risk profile
